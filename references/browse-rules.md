# Browse Rules

Use this reference when deciding how far to go on each website.

The goal is simple:

- collect as many useful public fields as possible for each relevant company
- if the homepage is incomplete, click into the next useful page
- if the site needs interaction such as tabs, buttons, cards, or pagination, continue browsing when the extra effort is worth it

## Core Principle

Start light, then go deeper only when needed.

Use this order:

1. search result snippet
2. landing page or homepage
3. one or two likely follow-up pages
4. deeper interactive browsing

Do not start with deep browsing unless there is a clear reason.

## What You Are Trying To Complete

The job is not only to find company names.

The job is to collect as much public information as possible for the requested output fields, so the exported file needs less manual review later.

Try to improve completion for fields such as:

- `CompanyName`
- `Website`
- `Tel`
- `Mail`
- `Address`
- `ContactPerson`
- `CoreProductOrMainBusiness`

## When The Homepage Is Enough

Stop at the current page only when it already gives strong field coverage and there is no obvious next page that is likely to add useful information.

Typical cases:

- the homepage clearly shows company name, products, and contact details
- the landing page is already a contact page
- the directory page already shows enough company details without extra clicks

## When To Click Into Another Page

If the homepage is relevant but incomplete, click into the next useful page.

High-priority follow-up pages:

1. `Contact` or `Contact Us`
2. `About` or `About Us`
3. `Locations`, `Branches`, or `Offices`
4. `Distributor Network`, `Dealer Network`, or `Sales Centers`
5. product or category pages when they help confirm the business type

Typical reasons to continue:

- homepage has company name but no phone or address
- homepage shows products but no contact information
- homepage looks like a brand page and the real contact details are one page deeper
- a directory result links to a detail page with fuller company information
- the homepage gives only partial company profile information and another likely page may reveal more fields

## When To Keep Browsing More Deeply

Go deeper when the website clearly hides useful information behind interaction.

Typical cases:

- contact details appear only after clicking tabs or cards
- the site uses buttons or pagination to show more results
- the page needs scrolling to load more companies
- supplier details appear only after opening a detail page
- the site is JavaScript-heavy and the plain page view is incomplete

This means the skill should be able to:

- click into another page
- click tabs or cards
- scroll to load more content
- move to the next page of results

## When To Stop

Stop and move on when:

- the page is clearly irrelevant
- the site is too weak to justify more clicks
- you have already reached the user's page limit or record-count target

## Practical Rule

For each result, quickly ask:

1. Is this company relevant?
2. How many useful fields do I already have from this page?
3. If not, is there an obvious next page such as `Contact` or `About`?
4. If not, is deeper browsing likely to reveal the missing fields?
5. Is the extra effort worth it?

If the current page already gives strong coverage and there is no obvious next step, stop.

If 2 is no and 3 is yes, click into the next useful page.

If 2 is no, 3 is no, and 4 is yes, continue deeper browsing.

If 2 is no, 3 is no, and 4 is no, skip and move on.
