## Fields

| Field | Description | Suggested Values / Fill Rules |
| --- | --- | --- |
| Region | 区域。公司业务区域、市场区域，或销售内部使用的区域标签。 | 可填写 `欧洲`, `欧亚交接`, `中东`, `东南亚` 等；优先沿用销售团队现有写法 |
| Country | 国家。公司所在国家，或网页/资料中显示的主要国家。 | 直接填写国家名称，如 `Turkey`, `Israel`, `Spain`, `China` |
| Category | 分类。线索分类、公司类型、产品分类，或销售内部标签。 | 可填写 `潜在`, `SCP`, 品类名，或其他内部约定值；优先沿用销售团队现有写法 |
| CompanyName | 公司名。主公司名、品牌名或跟进主体名称。 | 直接填写公司官方名称或常用商业名称 |
| Channel | 渠道。用于标记客户来源层级或渠道分类。 | 当前草稿值：`A`, `B1`, `B2`, `B3`, `C1`, `C2`, `C3` |
| Specialty | 专业。用于标记专业度、匹配度或内部分类等级。 | 当前草稿值：`1`, `2`, `3`, `4`, `5` |
| Scale | 规模。用于标记客户规模等级。 | 当前草稿值：`1`, `2`, `3`, `4`, `5` |
| CoreProductOrMainBusiness | 核心产品/主业。公司主营产品、核心产品线或主营业务。 | 当前草稿值：`综合泳池`, `热泵`, `水泵`, `盐机`, `太阳能`, `电池`, `变频器`, `其他` |
| EstablishedYear | 成立时间。公司成立年份或成立日期。 | 优先填写公开可验证的成立年份，如 `1998`；如果网页给出完整日期，也可保留原文 |
| EmployeeCount | 人数。公司员工人数或团队规模。 | 填公开人数信息，如 `50`, `200+`, `1,000-5,000`；未知留空 |
| AnnualRevenue | 营业额。公司公开营业额、年收入或营收区间。 | 填公开营收数据或区间，如 `$10M-$50M`、`EUR 20M`；未知留空 |
| ContactPerson | 联系人。公开联系人姓名、销售联系人、负责人，若没有则留空。 | 填公开联系人姓名，未知留空 |
| Mail | 邮箱。公开邮箱、销售邮箱或联系邮箱。 | 填公开邮箱地址，未知留空 |
| Tel | 电话。公开联系电话、公司总机或销售电话。 | 填公开电话号码，未知留空 |
| Website | 网页。官网首页或最相关的公司页面链接。 | 填 `https://...` 链接 |
| Address | 地址。公司地址、办公地址、工厂地址或公开联系地址。 | 优先填写网页公开地址；保留原网页语言和原始格式，未知留空 |
| Source | Source。线索来源。 | 当前草稿值：`网页`, `展会`, `介绍` |
| FollowUpStatus | 跟进状态。内部销售跟进状态标签。 | 当前草稿值：`S1: 建立联系`, `S2: 已报价`, `S3: 样机客户`, `S4: 订单客户`, `S5: 暂停`, `S6: 其他`, `S0: 无互动` |
