## Fields

| Field | Description | Suggested Values / Fill Rules |
| --- | --- | --- |
| Region | 公司注册区域 | 可填写 `欧洲`, `亚洲`, `中东`, `东南亚` 等 |
| Country | 公司所在国家，或网页/资料中显示的主要国家 | 直接填写国家名称，如 `Turkey`, `Israel`, `Spain`, `China` |
| CompanyName |主公司名、品牌名或跟进主体名称 | 直接填写公司官方名称或常用商业名称 |
| Channel | 该公司在销售链路中的商业角色，看它更接近生产商、总代、分销商、经销商，还是零售门店类渠道 | 默认值：`A:manufacturer`, `B1:master distributor `, `B2:distributor`, `B3:dealer`, `C1:online store`, `C2:chainstore`, `C3:pop store`, 询问客户情况之后，在输出的文件里，Channel 字段写A B1 B2 B3 C1 C2 C3 这些，在末尾标注上这些选项代表的意思。如果是其他需求，需要用户自定义，不同需求不同要求 |
| Specialty | 专业。用于标记专业度、匹配度或内部分类等级。 | 当前草稿值：泳池产品分销业务权重占比在80%以上`1：泳池业务权重占比低于50%`, `2:泳池设计，安装或维护业务为主`, `3：有做跨行产品例如空调或者暖通热泵air source heat pump 或者工业用热泵或水泵，而非泳池用且权重在50%或更高`, `4：有做跨行产品例如空调或者暖通热泵air source heat pump 或者工业用热泵或水泵，而非泳池用且权重低于50%`, `5 ：100%专注泳池设备分销或进口，而且分销的产品齐全，没有跨行产品` |
| Scale | 公司规模等级 | 当前草稿值：`1：年营业额低于300万欧元`, `2：年营业额在300万欧元或以上`, `3:营业额在500万欧元或以上；或者dealer在100-500个之间`, `4：营业额在1000万欧元或以上；或者dealer超过500个`, `5：营业在3000万欧元或以上或者dealer 超过1000个` 未知填'null' |
| CoreProductOrMainBusiness | 公司主营产品、核心产品线或主营业务 | 当前草稿值：`综合泳池设备`, `热泵`, `水泵`, `盐氯机`, `太阳能产品`, `储能电池`, `变频器`, `其他` |
| EstablishedYear | 公司成立年份或成立日期 | 优先填写公开可验证的成立年份，如 `1998`；如果网页给出完整日期，也可保留原文 |
| EmployeeCount | 公司员工人数或团队规模 | 填公开人数信息，如 `50`, `200+`, `1,000-5,000`；未知填'null' |
| AnnualRevenue | 公司公开营业额、年收入或营收区间 | 填公开营收数据或区间，如 `$10M-$50M`、`EUR 20M`；未知填'null' |
| ContactPerson | 公开联系人姓名、销售联系人、负责人，若未知填'null' | 填公开联系人姓名，未知填'null' |
| Mail | 公开邮箱、销售邮箱或联系邮箱 | 填公开邮箱地址，未知填'null' |
| Tel | 公开联系电话、公司总机或销售电话 | 填公开电话号码，未知填'null' |
| Website | 官网首页或最相关的公司页面链接 | 填 `https://...` 链接 |
| Address | 公司地址、办公地址、工厂地址或公开联系地址 | 优先填写网页公开地址；保留原网页语言和原始格式，未知填'null'|
| Source | 信息来源 | 例如：`网页`, `展会`, `介绍` |
| FollowUpStatus | 内部销售跟进状态 | 当前草稿值：`S1: 建立联系`, `S2: 已报价`, `S3: 样机客户`, `S4: 订单客户`, `S5: 暂停`, `S6: 其他`, `S0: 无互动` |
